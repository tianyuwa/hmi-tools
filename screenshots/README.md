# Product Screenshots

Place actual screenshots here for the product site.

## File naming convention

screenshot_<tool>.png

Where <tool> is one of:
- plc_mapping    — PLC Variable Auto-Mapping
- modbus_calc    — Modbus Address Calculator
- recipe_conv    — Recipe Data Converter
- tag_manager    — Tag Import/Export Tool
- io_report      — IO Point Table Report
- toolkit_bundle — Full Toolkit Bundle

## Recommended screenshot content

For each tool, provide 1-2 screenshots showing:
1. Input (what the user provides)
2. Output (what the tool generates)

Size: 1200x800px recommended
Format: PNG

## Current site

The site's "Demo" section uses HTML code samples as a substitute.
When real screenshots are available, add them with the naming above
and reference them in index.html.
